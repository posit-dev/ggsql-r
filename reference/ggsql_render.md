# Render a spec with a writer

This function takes a `Spec` object as returned by
[`ggsql_execute()`](https://r.ggsql.org/reference/ggsql_execute.md) and
renders it with the provided writer.

## Usage

``` r
ggsql_render(writer, spec)
```

## Arguments

- writer:

  A `Writer` object created by e.g.
  [`vegalite_writer()`](https://r.ggsql.org/reference/vegalite_writer.md).

- spec:

  A `Spec` object returned by
  [`ggsql_execute()`](https://r.ggsql.org/reference/ggsql_execute.md).

## Value

Writer dependent:

- `vegalite_writer`: A string holding the vegalite JSON representation
  of the visualization

## Examples

``` r
reader <- duckdb_reader()
ggsql_register(reader, mtcars, "cars")
spec <- ggsql_execute(reader,
  "SELECT * FROM cars VISUALISE mpg AS x DRAW histogram"
)

ggsql_render(vegalite_writer(), spec)
```
